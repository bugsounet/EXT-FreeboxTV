# EXT-FreeboxTV
## Plugin pour MMM-GoogleAssistant

EXT-Freebox est un plugin pour `MMM-GoogleAssistant`

Il permet d'afficher, sur votre Mirroir, les chaines de TV de la [Freebox](https://www.free.fr/freebox/).
